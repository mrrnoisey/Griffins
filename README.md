# Griffins

A minimal interactive landing page for @griffoh_123 with animated parallax particles, cursor trail, and click ripple effects.

## ✅ Ready for launch

- Single-page static site (`index.html`) with HTML, CSS, and JS in one file.
- Mobile responsive canvas with resize handling.
- Meta tags for SEO and theme color.
- Fallback and `noscript` support for graceful degradation.
- Accessible button and profile image alt text.

## 📦 Deploy

1. Commit and push to `main`.
2. For GitHub Pages: go to repository Settings > Pages, select `main` branch and `/ (root)` folder.
3. Use any static host (Netlify, Vercel, Cloudflare Pages), pointing to this repository.

## 🔧 Update

- Replace `your-image.jpg` in the project root or update the `img` `src` with hosted profile picture.
- Edit the `.bio` text and Instagram URL in `follow()`.

## 🧪 Test

- Open `index.html` locally in browser.
- Check mobile view and resize behavior
- Click FOLLOW button to confirm Instagram redirect.
